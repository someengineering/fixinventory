# resoto-plugin-github (WIP)
GitHub Collector Plugin for resoto

```
  --github-access-token GITHUB_ACCESS_TOKEN
                        GitHub access token
  --github-org GITHUB_ORGS [GITHUB_ORGS ...]
                        GitHub Organizations
  --github-repo GITHUB_REPOS [GITHUB_REPOS ...]
                        GitHub Repositories
  --github-user GITHUB_USERS [GITHUB_USERS ...]
                        GitHub Users
  --github-pool-size GITHUB_POOL_SIZE
                        GitHub Thread Pool Size (default: 5)
  --github-fork         GitHub use forked process instead of threads (default: False)
```
